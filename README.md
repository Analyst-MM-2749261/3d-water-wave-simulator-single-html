# 3D水面波形シミュレータ

ブラウザ上で動作する、単一HTML構成の3D水面波形シミュレータです。水面をタップして波紋を起こしたり、視点を回転・ズームしたりしながら、波の干渉や減衰を直感的に観察できます。

円形水槽の外周には造波弁が配置されており、入力した1〜5文字に対応するパルスを発射して、水面上に文字のような波形を一瞬だけ浮かび上がらせるデモも備えています。

## 主な機能

- 3D水面メッシュによるリアルタイム波動シミュレーション
- タップまたはクリックによる単発波紋の発生
- ランダム雨滴モード
- ワイヤーフレーム表示
- ドラッグによる視点回転
- ピンチ操作またはマウスホイールによるズーム
- 波高ヒートマップとエネルギー推移の簡易表示
- 外周造波弁による文字フラッシュ表示
- WebGL2対応環境でのGPU変位レンダリング
- WebGL2非対応環境でのCPU経路へのフォールバック

## 使い方

1. [公開ページ](https://analyst-mm-2749261.github.io/3d-water-wave-simulator-single-html/)をブラウザで開きます。
2. 水面をタップまたはクリックすると、波紋が発生します。
3. 画面をドラッグすると、視点を回転できます。
4. ピンチ操作またはマウスホイールでズームできます。
5. 画面下部の操作パネルから、雨、網目表示、波の速さ、減衰、波紋の強さなどを変更できます。
6. 「文字を浮かび上がらせる」に1〜5文字を入力し、「発射」を押すと、外周造波弁からパルスが発射されます。

## 動作環境

モダンブラウザでの利用を想定しています。PCブラウザとスマートフォンブラウザのどちらでも動作しますが、WebGLや端末性能により表示の滑らかさは異なります。

このアプリは単一HTMLファイルとして構成されており、実行時に外部CDNからJavaScriptライブラリを読み込まない形にしています。

## プライバシー

このアプリは静的なクライアントサイドアプリです。入力内容や操作内容をサーバーへ送信する機能はありません。
