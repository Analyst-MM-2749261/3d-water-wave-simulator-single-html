# Privacy Policy

## Summary

3D水面波形シミュレータは、ブラウザ内で動作する静的なクライアントサイドアプリです。

レビュー済みの公開用 `index.html` では、アプリ本体コードに `fetch`、`XMLHttpRequest`、`WebSocket`、`navigator.sendBeacon`、`localStorage`、`sessionStorage`、`indexedDB` は検出されていません。

## Data Collection

このアプリは、利用者の入力文字、操作内容、端末情報をアプリ制作者のサーバーへ送信しません。アプリ自体にサーバー通信機能はありません。

## Local Storage / Cookies

このアプリはCookie、LocalStorage、SessionStorage、IndexedDBを使いません。

## Text Input

「文字を浮かび上がらせる」機能で入力した文字列は、ブラウザ内で波形生成に使われます。レビュー済み公開用 `index.html` では、この文字列を外部へ送信する処理は検出されていません。

## GitHub Pages / Hosting Logs

GitHub Pagesや利用者のネットワーク環境では、一般的なアクセスログや通信メタデータがホスティング事業者・通信事業者側で処理される場合があります。このリポジトリのアプリ自体は、そのようなログを直接管理しません。

## Future Changes

アクセス解析、広告、外部API、問い合わせフォーム、サーバー送信、保存機能を追加する場合は、このプライバシーポリシーを更新してください。
